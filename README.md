# Demoblaze UI Tests

Playwright framework for the [Demoblaze Product Store](https://www.demoblaze.com).

## Requirements

- Node.js 18+

## Setup

```bash
npm install
npx playwright install chromium
```

## Running the tests

Run all tests:

```bash
npm test
```

Run a specific file:

```bash
npx playwright test tests/part1-product-scraping.spec.js
npx playwright test tests/part2-purchase-flow.spec.js
npx playwright test tests/bonus.spec.js
```

Open the HTML report after a run:

```bash
npm run report
```

## What each file does

**`tests/part1-product-scraping.spec.js`**
Navigates the first two pages of the store, extracts name, price, and link of every product, and saves the result to `products.txt`.

**`tests/part2-purchase-flow.spec.js`**
- Completes a full purchase (product → cart → checkout → confirmation)
- Filters products by category and validates the result
- Adds a product to the cart and removes it

**`tests/bonus.spec.js`**
- Registers a unique test user (timestamp-based, no collision risk)
- Tries login with empty fields → expects error
- Tries login with wrong password → expects error
- Logs in with valid credentials → verifies welcome message
- Logs out → verifies session is cleared
- Submits the contact form

## Project structure

```
src/
├── fixtures.js       # Injects page objects into tests
└── pages/
    ├── home.page.js      # Listing, pagination, category filter, contact modal
    ├── product.page.js   # Product detail, add to cart
    ├── cart.page.js      # Cart items, checkout, clear cart
    └── login.page.js     # Sign up, login, logout
tests/
├── part1-product-scraping.spec.js
├── part2-purchase-flow.spec.js
└── bonus.spec.js
products.txt          # Generated by part1 test
```
